# CSS简介

#### 用于定义网页样式和布局的样式语言

#### 通过CSS可以指定页面中 个元素的颜色，字体，大小，间距，边框，背景等样式，从而实现更精确的页面设计

# CSS语法

1. #### 选择器中的声明可以写无数条属性

2. #### 声明的每一行属性，都要以英文分号结尾;

3. #### 声明中的所有属性和值都是以键值对的形式出现的

# CSS三种导入方式

### 注意：三种方式有优先级之分，内联样式>内部样式>外部样式，优先级高的会覆盖掉优先级低的样式

1. #### 内联样式：直接放入HTML标签内

   #### 用以下格式

   #### \<h1 style="color: blue; height: 16px">\<h1>

   #### 可以包含多个样式，但要包含在一个style中

2. #### 内部样式：放在HTML的head标签中

   #### 在head标签中创建一个\<style>\</style>在其中写入CSS，style中的格式为

   #### p{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### 作用是使所有含有\<p>标签的文字都编成蓝色且大小变为16px

3. #### 外部样式：把CSS样式单独放在一个CSS文件中,再在HTML的head标签中用另外一个标签链接进来，便于在多个页面上使用多个样式

   #### 在vscode中，在本HTML文件的头目录下创建一个创建一个CSS文件夹，再在CSS文件夹里创建一个后缀为.css的文件，里面的格式就如2.内部样式style中的格式一致

   #### 然后在HTML标签中head中加入\<link rel="stylesheet" herf="文件路径"

# CSS选择器

#### 定义：用于选择要应用样式的HTML元素

1. #### 元素选择器：就是笔记上方所提到的选择器

   #### p{

   #### color: blue;

   #### font-size: 16px;

   #### }

2. #### 类选择器：选择指定`类`的元素

   #### .highlight{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### highlight就是类名，在前方加上一个.

3. #### id选择器：

   #### #header{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### header是id，前方加一个#

4. #### 通用选择器：

   #### *{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### 前方加一个*就可以选择全部元素

5. #### 子元素选择器：选择父元素内部的元素（嵌套），此选择器只适用于嵌套一层的元素

   #### .father> .son{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### father是父元素的类名，son是子元素的类名

6. #### 后代选择器：选择父元素中的子元素，不管嵌套了多少层

   #### .father .son{

   #### color: blue;

   #### font-size: 16px;

   #### }

7. #### 兄弟选择器：选择相邻的元素

   #### h3 + p{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### h3是主元素，p是主元素相邻的元素，一般优先选择位于主元素下方的元素

8. #### 伪类选择器：应用于交互

   #### #highlight:hover{

   #### color: blue;

   #### font-size: 16px;

   #### }

   #### highlight是类名，hover是交互的样式

9. #### 伪元素选择器：用::开头插入虚拟的元素

# CSS常见属性

1. #### font：在一个声明中设置所有字体属性，这是一个复合属性

2. #### line-height：调整行间距

3. #### width与height：只有块级元素和行内块元素(图片)才能使用此属性；width改变宽度，height改变高度；图片用width是按比例改变大小

4. #### 用display属性可以实现块级，行内，，行内块元素的转换

   #### bolck块级，inline行内，inline-block行内块

5. #### border：给元素生成边框，这是一个复合属性

   #### border: 5px solid red;生成一个宽度为5px，红色的(red)，实线(solid)边框

   #### solid实线，dashed虚线，double双实线，dotted点线

# 浮动 

#### 定义：用于创建浮动框，将其移动到一边，直到左边缘或右边缘初级包含块或另一个浮动框的边缘，这样即可使得元素进行浮动

1. #### 选择器{

   ####           float: left/right/none;

   #### }

   #### 注意：浮动是相对于父元素浮动，只会在父元素内部移动

2. #### 应用：可以让多个块级元素在一行内显示，且块级元素中间没有缝隙

3. #### 浮动的清除：若没有给父元素设置高度则子元素浮动过后，在添加文本会被浮动覆盖，此时就需要消除浮动

   #### 两种方式：

   1. #### 在父元素内加上overflow: hidden;

   2. #### 伪元素清除法：在头文件中添加

      #### .父元素类名::after{

      #### content:"";

      #### display:table;

      #### clear:both;

      #### }

# 定位

- #### 相对定位：只改变相应元素的位置其他元素位置不变

- #### 绝对定位：相对于已经定位的父级元素定位，若没有已定位的父级元素就相当于HTML文档定位，改变相应元素时，在其后方的元素会自动补位

- #### 固定定位：相对于浏览器窗口进行定位

  #### 在相应元素选择器中添加position:relative(相对)/absolute(绝对)/fixed(固定)
